白手起家建立 Hugo Blog

我執起網絡日誌，自己建、自己寫，免除担心24字管理去限制我。這種獨立網誌也是私人財產，無需驚青1博客服務商搿硬2沒收。不過，獨立網誌也有代價。要生態上網，要識得最少一種西文，要定得落嚟解決問題。雖然麻煩，多種日誌搭建平臺裏，我揀咗 Hugo Framework。

簡介 Hugo

  • 快

The world’s fastest framework for building websites

– gohugo.io

  • 沙盒 以 git bash 命令 hugo server -w 完成本地調試，預覽頁面樣式無誤後再決定正式發佈。

  • 專注 使用 Markdown 語句寫作，視線幾乎不咧開屏幕。

材料、工具

  • Windows 10，或者其他 OS
  • git
  • hugo
  • notion.so

安裝 git 或者 hugo 撞到異常，記落Notion。唔介意手抄，白紙又得。

優化

安裝 git 、hugo 後，下載 Hugo 網站樣式，如果一切順利，可以啓動沙盒靜觀 hugo theme。

  • 首頁僅顯示日誌標題（已完成）
  • 首頁 index.html 與日誌頁 single.html區別顯示（已完成）
  • 根據 tags ，日誌頁加載 related posts（計畫）
  • 裝載評論系統（計畫）

一鍵發佈或更新日誌

趙化冰 Hugo Theme Deanwhite 提供咗一段 .sh 代碼，鼠標一點就更新網站：

#!/bin/bash

echo -e "\033[0;32mDeploying updates to GitHub...\033[0m"

# Build the project.
hugo # if using a theme, replace with `hugo -t <YOURTHEME>`

# Go To Public folder
cd public
# Add changes to git.
git add .

# Commit changes.
msg="rebuilding site `date`"
if [ $# -eq 1 ]
  then msg="$1"
fi
git commit -m "$msg"

# Push source and build repos.
git push origin master

# Come Back up to the Project Root
cd ..

同理，如果要一 click 睇沙盒下日誌，你可以建立 server.sh，內容寫入：

#!/bin/bash

echo -e "\033[0;32mgenerate local server...\033[0m"

# activate server mode
hugo server -w

cd ..
